## About WebStack

WebStack is a navigation website for designers, collecting excellent design websites, UI design resources, and inspiration websites from around the world, providing a one-stop resource navigation service for designers.

## Usage

- Browse different categories through the left sidebar menu
- Use the search box at the top to quickly find websites
- Click the icon in the top right to toggle dark mode
- Click any card to visit the corresponding website

## Technology

This project is built with Hugo static site generator and Decap CMS content management system.

- Frontend: Bootstrap 3 + Xenon Theme
- Site Generator: Hugo
- Content Management: Decap CMS
- Deployment: GitHub Pages + GitHub Actions
