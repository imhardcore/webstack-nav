## 关于 WebStack

WebStack 是一个面向设计师的网址导航网站，收集整理了国内外优秀的设计网站、UI设计资源、灵感创意网站等，为设计师提供一站式资源导航服务。

## 使用说明

- 通过左侧菜单浏览不同分类的网站
- 使用顶部搜索框快速查找网站
- 点击右上角图标切换暗色模式
- 点击任意卡片即可访问对应网站

## 技术支持

本项目基于 Hugo 静态站点生成器和 Decap CMS 内容管理系统构建。

- 前端框架: Bootstrap 3 + Xenon Theme
- 站点生成: Hugo
- 内容管理: Decap CMS
- 部署: GitHub Pages + GitHub Actions
